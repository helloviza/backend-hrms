# Cert Inventory Snapshots

Snapshots of TBO inventory mapping for the cert account, generated by `pnpm cert:inventory`.

## Files

- `inventory-latest.json` / `inventory-latest.md` — most recent snapshot, overwritten on each run.
- `inventory-<ISO timestamp>.json` / `.md` — historical snapshots.

## When to regenerate

- Before sending an email to TBO about inventory.
- After TBO confirms they've added inventory mapping (to verify).
- For cert evidence package.

## Run

```
cd apps/backend
pnpm cert:inventory          # default scope: 20 cert-relevant countries
pnpm cert:inventory -- --full # full traversal of all 249 countries (slow, hours)
```

## Source

Counts are pulled live from TBO's `HotelCodeList` API. This file is the source of truth; do not hand-edit. To update, re-run the script.
